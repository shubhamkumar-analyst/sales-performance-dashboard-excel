# Sales Performance Dashboard (Excel)

## Project Overview
**Sales Performance Dashboard (Excel)**

This project is an interactive Excel dashboard designed to analyze overall sales performance and customer behavior using dynamic visualizations. The dashboard provides insights into top-performing states, product categories, and customer engagement metrics.

Key metrics such as **Repeat Customer Percentage**, **Customer Count**, and **Average Revenue per Customer** help evaluate customer loyalty and business growth. The dashboard also includes a **Dynamic Insight Box** that automatically highlights important business trends based on the selected filters.

Using pivot tables, pivot charts, and slicers, the dashboard allows users to explore sales data interactively and quickly identify patterns, high-performing regions, and potential improvement areas.

**Key Features**
• Sales performance overview
• Top performing states and product categories
• Repeat Customer % analysis
• Customer count tracking
• Average revenue per customer
• Dynamic insight box for quick business insights
• Interactive filters using slicers

**Tools Used:** Microsoft Excel, Pivot Tables, Pivot Charts, Data Cleaning, Slicers
